隐私说明

数据如何使用

当你主动触发翻译时，扩展会读取当前网页中可见的文本内容，并发送到你在设置页配置的 OpenAI 兼容 API 服务，用于生成译文。

扩展不会出售、共享或上传你的 API Key。API Key 保存在本机浏览器扩展存储中，仅用于向你配置的 API 服务发起请求。

本地缓存

默认不保存译文缓存。若你在设置页启用本地译文缓存，译文会保存在本机浏览器扩展存储中，以减少重复 API 请求。

你可以在扩展设置页关闭缓存，或在浏览器扩展详情/站点数据设置中清除扩展存储。

第三方 API

请只配置你信任的 HTTPS API 服务。第三方 API 服务如何处理请求内容，取决于该服务自己的隐私政策和服务条款。
